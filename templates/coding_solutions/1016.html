Given a binary string S (a string consisting only of '0' and '1's) and a positive integer N, return true if and only if for every integer X from 1 to N, the binary representation of X is a substring of S.

{% endblock question %} {% block solution_exp %}

Generate binary strings for a given integer N using helper function for all values i where 1 <= i <= N.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: bool queryString(string S, int N) { bool contains = true; for (int i = 1; i <= N; i++){ string bin = ""; binary(i, bin); if (S.find(bin) == string::npos){ contains = false; break; } } return contains; } void binary(int N, string& bin){ if (N > 1) binary(N/2, bin); if (N % 2 > 0) bin += '1'; else bin += '0'; } }; {% endblock solution_code %}

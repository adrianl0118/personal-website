Given a valid (IPv4) IP address, return a defanged version of that IP address.

A defanged IP address replaces every period "." with "[.]".

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: string defangIPaddr(string address) { stringstream rewrite; for (int i = 0; i < address.size(); i++){ if (address[i] == '.') rewrite << "[.]"; else rewrite << address[i]; } return rewrite.str(); } }; {% endblock solution_code %}

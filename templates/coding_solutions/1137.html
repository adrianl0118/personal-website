The Tribonacci sequence Tn is defined as follows:

T0 = 0, T1 = 1, T2 = 1, and Tn +3 = Tn + Tn+1 + Tn+2 for n >= 0.

Given n, return the value of Tn.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int tribonacci(int n) { int x=0, y=1,z=1, len=2; vector<int> tribs{0,1,1}; while(n > len){ tribs.push_back(x+y+z); int old=x; x=y; y=z; z=x+y+old; len++; } return tribs[n]; } }; {% endblock solution_code %}

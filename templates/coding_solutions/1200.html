Given an array of distinct integers arr, find all pairs of elements with the minimum absolute difference of any two elements.

Return a list of pairs in ascending order(with respect to pairs), each pair [a, b] follows

  • a, b are from arr
  • a < b
  • b - a equals to the minimum absolute difference of any two elements in arr
{% endblock question %} {% block solution_exp %}

Sort vector (O(nlogn)) and then use loop iterations to A) Find the minimum absolute difference and B) Store all the viable pairs.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector> minimumAbsDifference(vector& arr) { sort(arr.begin(),arr.end()); int MinDiff = 200000; for (int i = 1; i < arr.size(); i++){ MinDiff = min(MinDiff,arr[i]-arr[i-1]); } vector> pairs; for (int i = 1; i < arr.size(); i++){ if (arr[i]-arr[i-1] == MinDiff){ pairs.push_back({arr[i-1],arr[i]}); } } return pairs; } }; {% endblock solution_code %}

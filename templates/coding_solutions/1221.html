Balanced strings are those who have equal quantity of 'L' and 'R' characters.

Given a balanced string s split it in the maximum amount of balanced strings.

Return the maximum amount of splitted balanced strings.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int balancedStringSplit(string s) { int lcount = 0; int rcount = 0; int balcount = 0; for (int i = 0; i < s.size(); i++){ if (s[i] == 'R'){ rcount++; } else { lcount++; } if (rcount == lcount){ rcount = 0; lcount = 0; balcount++; } } return balcount; } }; {% endblock solution_code %}

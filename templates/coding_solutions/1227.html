n passengers board an airplane with exactly n seats. The first passenger has lost the ticket and picks a seat randomly. But after that, the rest of passengers will:

  • Take their own seat if it is still available,
  • Pick other seats randomly when they find their seat occupied

What is the probability that the n-th person can get his own seat?

{% endblock question %} {% block solution_exp %}

There's a 50% chance that any passenger other than the guy who lost his seat will have their seat occupied by that guy. i.e. did it happen or didn't it? If the plane only has 1 person, then that person will definitely get their own seat.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: double nthPersonGetsNthSeat(int n) { return (n == 1)? 1.0 : 0.5; } }; {% endblock solution_code %}

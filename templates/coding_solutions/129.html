Given a binary tree containing digits from 0-9 only, each root-to-leaf path could represent a number.

An example is the root-to-leaf path 1->2->3 which represents the number 123.

Find the total sum of all root-to-leaf numbers.

Note: A leaf is a node with no children.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int sumNumbers(TreeNode* root) { if (!root) return 0; int sum = 0; string path = ""; DFS(root, path, sum); return sum; } void DFS(TreeNode* root, string path, int& sum){ path += to_string(root->val); if (!root->left && !root->right){ if (path.size()) sum += stoi(path); return; } if (root->left) DFS(root->left, path, sum); if (root->right) DFS(root->right, path, sum); return; } }; {% endblock solution_code %}

Given an array arr, replace every element in that array with the greatest element among the elements to its right, and replace the last element with -1.

After doing so, return the array.

{% endblock question %} {% block solution_exp %}

Iterating over array front to back, at each index use std::max_element on the range of the array to the right of the current index.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector<int> replaceElements(vector<int>& arr) { for (int i = 0; i < arr.size()-1; i++) arr[i] = *max_element(arr.begin()+i+1, arr.end()); arr[arr.size()-1] = -1; return arr; } }; {% endblock solution_code %}

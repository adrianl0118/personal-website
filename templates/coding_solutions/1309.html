Given a string s formed by digits ('0' - '9') and '#' . We want to map s to English lowercase characters as follows:

  • Characters ('a' to 'i') are represented by ('1' to '9') respectively.
  • Characters ('j' to 'z') are represented by ('10#' to '26#') respectively.

Return the string formed after mapping.

It's guaranteed that a unique mapping will always exist.

{% endblock question %} {% block solution_exp %}

Iterate from back to front to identify the character '#' and parse accordingly. ASCII '0' starts at an integer value of 48--use this with some char arithmetic to translate for the character from the code.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: string freqAlphabets(string s) { string ans = ""; for (int i = s.size()-1; i >= 0; i--){ char letter; if (s[i] == '#'){ letter = ((s[i-2]-48)*10+s[i-1]-48)+'a'-1; ans = letter + ans; i-=2; } else { letter = (s[i]-48)+'a'-1; ans = letter + ans; } } return ans; } }; {% endblock solution_code %}

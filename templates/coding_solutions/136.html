Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

{% endblock question %} {% block solution_exp %}

XOR operation of two identical numbers is zero and XOR of zero and another number is the non-zero number, XOR all the numbers together and you'll be left with only the unique value.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int singleNumber(vector<int>& nums) { int a = 0; for (int& n : nums){ a ^= n; } return a; } }; {% endblock solution_code %}

There is a pizza with 3n slices of varying size, you and your friends will take slices of pizza as follows:

  • You will pick any pizza slice.
  • Your friend Alice will pick next slice in anti clockwise direction of your pick.
  • Your friend Bob will pick next slice in clockwise direction of your pick.
  • Repeat until there are no more slices of pizzas.

Sizes of Pizza slices are represented by circular array slices in clockwise direction.

Return the maximum possible sum of slice sizes which you can have.

{% endblock question %} {% block solution_exp %}

Break up the circular problem into two linear problems, one ranging from the first slice to the second last slice and the other from the second slice to the last slice (if we pick the first slice, we can't have the last slice because it is adjacent and vice versa).

For each linear problem, iterate through all the possible slices. Maintain a DP vector where each row represents a possible sequence of pizza slices eaten by the person, starting from some index in the slices linear array. In the DP, index 2 will correspond to index 0 in the slices array so that we can incrementally, uni-directionally calculate the total amount of pizza eaten; at any slice, the total eaten will be the maximum of eating the current slice and the one two slices before it or eating the previous slice.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int maxSizeSlices(vector& slices) { int n = (int)slices.size() / 3; auto l1 = vector(slices.begin(), slices.end()-1); //two problems, can't take first slice and last at the same time auto l2 = vector(slices.begin()+1, slices.end()); return max(linear(l1, n), linear(l2, n)); } private: int linear(vector& slices, int n) { vector> eat((int)slices.size()+2, vector(n+1, INT_MIN)); //dp vector, each vector represents a possible sequence of pizza slice sizes for the person to eat int ans = INT_MIN; for (int i = 0; i < eat.size(); i++) eat[i][0] = 0; for (int i = 2; i < eat.size(); i++) { for (int j = 1; j < n + 1; j++) //for every pizza slice in this eating, eat[i][j] = max(eat[i-1][j], eat[i-2][j-1] + slices[i-2]); //take maximum of eating current slice and two slices before that or eating the previous slice (which precludes being able to eat the current slice) ans = max(eat[i][n], ans); } return ans; } }; {% endblock solution_code %}

Given the array nums, obtain a subsequence of the array whose sum of elements is strictly greater than the sum of the non included elements in such subsequence.

If there are multiple solutions, return the subsequence with minimum size and if there still exist multiple solutions, return the subsequence with the maximum total sum of all its elements. A subsequence of an array can be obtained by erasing some (possibly zero) elements from the array.

Note that the solution with the given constraints is guaranteed to be unique. Also return the answer sorted in non-increasing order.

{% endblock question %} {% block solution_exp %}

Sort numbers to allow for partial iteration as opposed iterating over the entire set to find sub sequences. From greatest to least, accumulate a sum until that sum exceeds half of the total sum of the array, storing the sum at each iteration. Sort the found sums from greatest to least and return

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector minSubsequence(vector& nums) { sort(nums.begin(),nums.end()); int sum = accumulate(nums.begin(),nums.end(),0); vector ans; while (accumulate(ans.begin(),ans.end(),0) <= sum/2){ ans.push_back(nums.back()); nums.pop_back(); } sort(ans.begin(),ans.end(),greater()); return ans; } }; {% endblock solution_code %}

You are given the array paths, where paths[i] = [cityAi, cityBi] means there exists a direct path going from cityAi to cityBi. Return the destination city, that is, the city without any path outgoing to another city.

therefore, there will be exactly one destination city.

{% endblock question %} {% block solution_exp %}

Map all origins to destinations. Iterate over destinations to determine the location that is not mapped to a destination.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: string destCity(vector<vector<string>>& paths) { unordered_map<string,string> dest; for (auto& p : paths){ dest[p[0]] = p[1]; } for (auto& p : paths){ if (dest.count(p[1]) == 0 ){ return p[1]; } } return ""; } }; {% endblock solution_code %}

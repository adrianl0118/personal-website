Given an array target and an integer n. In each iteration, you will read a number from list = {1,2,3..., n}.

Build the target array using the following operations:

  • Push: Read a new element from the beginning list, and push it in the array.
  • Pop: delete the last element of the array.
  • If the target array is already built, stop reading more elements.

You are guaranteed that the target array is strictly increasing, only containing numbers between 1 to n inclusive.

Return the operations to build the target array.

You are guaranteed that the answer is unique.

{% endblock question %} {% block solution_exp %}

Maintain a pointer on the target array, iterate in increasing order concurrently with iterating 1 to n. Place numbers that are in target with "Push". If they aren't in target, just "Push" and then immediately "Pop".

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector<string> buildArray(vector<int>& target, int n) { vector<string> ans; int idx = 0; for (int i = 1; i <= n; i++){ if (idx == target.size()) break; if (i == target[idx]){ ans.push_back("Push"); idx++; } else { ans.push_back("Push"); ans.push_back("Pop"); } } return ans; } }; {% endblock solution_code %}

Given a string s, the power of the string is the maximum length of a non-empty substring that contains only one unique character.

Return the power of the string.

{% endblock question %} {% block solution_exp %}

Maintain tracking of previous letter while iterating through array. If current char is the same as previous, increment count else reset count to zero.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int maxPower(string s) { char prev = '-'; int count = 0; int mcount = 0; for (auto& c : s){ count = (c == prev)? count+1 : 1; //Count if a repeat, otherwise reset count mcount = max(mcount,count); //Keep track of the biggest count prev = c; } return mcount; } }; {% endblock solution_code %}

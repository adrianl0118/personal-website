Given the array of integers nums, you will choose two different indices i and j of that array. Return the maximum value of (nums[i]-1)*(nums[j]-1).

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int maxProduct(vector<int>& nums) { sort(begin(nums),end(nums)); return (nums.back()-1)*(nums[nums.size()-2]-1); } }; {% endblock solution_code %}

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.

You may assume that the array is non-empty and the majority element always exist in the array.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int majorityElement(vector<int>& nums) { unordered_map<int,int> numcount; //count occurrences of numbers in nums for (auto& n : nums) numcount[n]++; int ans = -1; for (auto& m : numcount) if (m.second > nums.size()/2){ ans = m.first; break; } return ans; } }; {% endblock solution_code %}

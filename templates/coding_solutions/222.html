Given a complete binary tree, count the number of nodes.

Note: Definition of a complete binary tree from Wikipedia:
In a complete binary tree every level, except possibly the last, is completely filled, and all nodes in the last level are as far left as possible. It can have between 1 and 2h nodes inclusive at the last level h.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int countNodes(TreeNode* root) { return !root ? 0 : 1 + countNodes(root->left) + countNodes(root->right); } }; {% endblock solution_code %}

Given a binary search tree, write a function kthSmallest to find the kth smallest element in it.

Note: You may assume k is always valid, 1 ≤ k ≤ BST's total elements.

{% endblock question %} {% block solution_exp %}

Traverse tree to store all values, sort values and access (k-1)th element.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int kthSmallest(TreeNode* root, int k) { vector smallest; storeValues(root, smallest); sort(smallest.begin(),smallest.end()); return smallest[k-1]; } void storeValues(TreeNode* root, vector& smallest){ if (!root) return; smallest.push_back(root->val); storeValues(root->left, smallest); storeValues(root->right, smallest); return; } }; {% endblock solution_code %}

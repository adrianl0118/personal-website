Write a function to delete a node (except the tail) in a singly linked list, given only access to that node.

{% endblock question %} {% block solution_exp %}

Make the pointer to the current node point to the next node, effectively removing the current node from the list.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: void deleteNode(ListNode* node) { *node = *(node->next); return; } }; {% endblock solution_code %}

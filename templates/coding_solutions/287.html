Given an array nums containing n + 1 integers where each integer is between 1 and n (inclusive), prove that at least one duplicate number must exist. Assume that there is only one duplicate number, find the duplicate one.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int findDuplicate(vector<int>& nums) { sort(begin(nums),end(nums)); for (int i = 1; i < nums.size(); i++) if (nums[i] == nums[i-1]) return nums[i]; return -1; } }; {% endblock solution_code %}

Given a non negative integer number num. For every number i in the range 0 ≤ i ≤ num calculate the number of 1's in their binary representation and return them as an array.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector<int> countBits(int num) { vector<int>dp(num+1,0); dp[0] = 0; for(int i = 1; i <= num; i++) dp[i] = dp[i>>1] + i % 2; return dp; } }; {% endblock solution_code %}

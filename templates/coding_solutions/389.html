Given two strings s and t which consist of only lowercase letters.

String t is generated by random shuffling string s and then add one more letter at a random position.

Find the letter that was added in t.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: char findTheDifference(string s, string t) { unordered_map<char,int> lc; for (auto& c : s) lc[c]++; for (auto& c : t) lc[c]--; for (auto& p : lc) if (p.second != 0) return p.first; return ' '; } }; {% endblock solution_code %}

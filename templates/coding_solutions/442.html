Given an array of integers, 1 ≤ a[i] ≤ n (n = size of array), some elements appear twice and others appear once.

Find all the elements that appear twice in this array.

Could you do it without extra space and in O(n) runtime?

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector findDuplicates(vector& nums) { vector ans; for (int i = 0; i < nums.size(); i++){ int index = abs(nums[i])-1; if (nums[index] < 0){ ans.push_back(abs(nums[i])); } nums[index] = -nums[index]; } return ans; } }; {% endblock solution_code %}

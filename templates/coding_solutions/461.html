The Hamming distance between two integers is the number of positions at which the corresponding bits are different.

Given two integers x and y, calculate the Hamming distance.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int hammingDistance(int x, int y) { unsigned int z = x^y; //x is positive unsigned int count = 0; while (z) { count += z & 1; //n & 1 = 1 if the last digit is 1 -- we count if this is the case z >>= 1; //right shift, we examine the next digit up until all bits gone } return count; } }; {% endblock solution_code %}

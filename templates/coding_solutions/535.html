TinyURL is a URL shortening service where you enter a URL such as https://leetcode.com/problems/design-tinyurl and it returns a short URL such as http://tinyurl.com/4e9iAk.

Design the encode and decode methods for the TinyURL service. There is no restriction on how your encode/decode algorithm should work. You just need to ensure that a URL can be encoded to a tiny URL and the tiny URL can be decoded to the original URL.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: string base="http://tinyurl.com/"; unordered_map<string,string> shorttolong; // Encodes a URL to a shortened URL. string encode(string longUrl) { string s = to_string(hash<string>{}(longUrl)); shorttolong[base+s]=longUrl; return base+s; } // Decodes a shortened URL to its original URL. string decode(string shortUrl) { return shorttolong[shortUrl]; } }; {% endblock solution_code %}

Given a string, you need to reverse the order of characters in each word within a sentence while still preserving whitespace and initial word order.

{% endblock question %} {% block solution_exp %}

Iterate through string character by character and use a stack to keep track of the most recent non-white space characters, accessible in reverse. Produce reverse word by emptying stack whenever white space is encountered and when the end of the string is reached.

Not as space efficient as in-place solutions as a new string is concurrently built during the iteration. O(2n)

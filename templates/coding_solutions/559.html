Given a n-ary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Nary-Tree input serialization is represented in their level order traversal, each group of children is separated by the null value (See examples).

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int Max_Depth = 0; int maxDepth(Node* root) { if (!root){ return 0; } int curr_depth = 0; CalculateAndCompareDepth(root,curr_depth); return Max_Depth; } void CalculateAndCompareDepth(Node* root, int curr_depth){ if (root){ curr_depth++; Max_Depth = max(Max_Depth, curr_depth); for (int i = 0; i < root->children.size(); i++){ CalculateAndCompareDepth(root->children[i], curr_depth); } } } }; {% endblock solution_code %}

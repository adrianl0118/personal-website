Given an array of integers and an integer k, you need to find the total number of continuous subarrays whose sum equals to k.

{% endblock question %} {% block solution_exp %}

Naive iteration sum. Speed could be improved by using a prefix sum instead.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int subarraySum(vector<int>& nums, int k) { int count = 0; for (int i = 0; i < nums.size(); i++){ int sum = 0; for (int j = i; j < nums.size(); j++){ sum += nums[j]; if (sum == k) count ++; } } return count; } }; {% endblock solution_code %}

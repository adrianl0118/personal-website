Given an n-ary tree, return the preorder traversal of its nodes' values.

Nary-Tree input serialization is represented in their level order traversal, each group of children is separated by the null value (See examples).

Follow Up: Recursive solution is trivial, could you do it iteratively?

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector<int> preorder(Node* root) { vector<int> result; dive(result, root); return result; } void dive (vector<int>& tab, Node* root){ if (root == NULL){ return; } else { tab.push_back(root->val); for (auto i: root->children){ if (i != NULL){ dive (tab,i); } } } return; } }; {% endblock solution_code %}

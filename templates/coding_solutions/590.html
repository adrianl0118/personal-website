Given an n-ary tree, return the postorder traversal of its nodes' values.

Nary-Tree input serialization is represented in their level order traversal, each group of children is separated by the null value (See examples).

Follow up: Recursive solution is trivial, could you do it iteratively?

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector<int> postorder(Node* root) { vector<int> result; dip(result, root); return result; } void dip(vector<int>& tab, Node* root){ if(root == NULL){ return; } else { for (auto i: root->children){ if (i != NULL){ dip(tab, i); } } tab.push_back(root->val); } return; } }; {% endblock solution_code %}

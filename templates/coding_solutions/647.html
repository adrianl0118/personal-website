Given a string, your task is to count how many palindromic substrings are in this string.

The substrings with different start indexes or end indexes are counted as different substrings even they consist of same characters.

{% endblock question %} {% block solution_exp %}

For all letters in the string, symmetrically expand the field of view about each letter and count palindromes obsered until there aren't anymore.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int countSubstrings(string s) { int count = 0; for (int i = 0; i < s.size(); i++){ count++; int left = i-1; int right = i; while (left >= 0 && right < s.size()){ if (s[left] == s[right]) count++; else break; left--; right++; } left = i-1; right = i+1; while (left >= 0 && right < s.size()){ if (s[left] == s[right]) count++; else break; left--; right++; } } return count; } }; {% endblock solution_code %}

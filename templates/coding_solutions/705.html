Design a HashSet without using any built-in hash table libraries.

To be specific, your design should include these functions:

  • add(value): Insert a value into the HashSet.
  • contains(value) : Return whether the value exists in the HashSet or not.
  • remove(value): Remove a value in the HashSet. If the value does not exist in the HashSet, do nothing.
{% endblock question %} {% block solution_exp %}

Using boolean vector. Optimizes speed at the expense of memory (98% Speed, 67% Memory)

{% endblock solution_exp %} {% block solution_code %} class MyHashSet { public: vector<bool> contents = vector<bool>(1000001,false); /** Initialize your data structure here. */ MyHashSet() { } void add(int key) { contents[key] = true; } void remove(int key) { if (contents[key] = true){ contents[key] = false; } } /** Returns true if this set contains the specified element */ bool contains(int key) { return contents[key]; } }; {% endblock solution_code %}

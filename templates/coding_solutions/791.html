S and T are strings composed of lowercase letters. In S, no letter occurs more than once.

S was sorted in some custom order previously. We want to permute the characters of T so that they match the order that S was sorted. More specifically, if x occurs before y in S, then x should occur before y in the returned string.

Return any permutation of T (as a string) that satisfies this property.

{% endblock question %} {% block solution_exp %}

Use an unordered_map to keep track of all available letters in T. Iterating over S, append all instances of the letter that are in T. Iterating over the map, append all leftover letters to the end of the answer string and return.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: string customSortString(string S, string T) { string ans = ""; unordered_map<char,int> letters; for (auto c : T){ letters[c]++; } for (auto s : S){ while (letters[s] > 0){ ans += s; letters[s]--; } } for (auto l : letters){ while (l.second > 0){ ans += l.first; l.second--; } } return ans; } }; {% endblock solution_code %}

Given a string S and a character C, return an array of integers representing the shortest distance from the character C in the string.

{% endblock question %} {% block solution_exp %}

Not ASCII differences, but physical distance within the string. Do one pass front to back and one back to front and take the minimum observed distance of a given character from the given character in either direction.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: vector shortestToChar(string S, char C) { int prev = INT_MIN; vector result; for (int i = 0; i < S.size(); i++){ if (S[i] == C){ prev = i; } if (prev == INT_MIN){ result.push_back(INT_MAX); } else { result.push_back(i - prev); } } prev = INT_MAX; for (int i = S.length()-1; i >= 0; i--) { if (S[i] == C){ prev = i; } result[i] = min(result[i], prev-i); } return result; } }; {% endblock solution_code %}

Given two strings S and T, return if they are equal when both are typed into empty text editors. # means a backspace character.

Note that after backspacing an empty text, the text will continue empty.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: bool backspaceCompare(string S, string T) { string ref = ""; for (auto& s : S){ if (s == '#'){ if (!ref.empty()) ref.pop_back(); } else ref.push_back(s); } string tref = ""; for (auto& t : T){ if (t == '#'){ if (!tref.empty()) tref.pop_back(); } else tref.push_back(t); } return ref == tref; } }; {% endblock solution_code %}

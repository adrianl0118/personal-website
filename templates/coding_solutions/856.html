Given a balanced parentheses string S, compute the score of the string based on the following rule:

  • () has score 1
  • AB has score A + B, where A and B are balanced parentheses strings.
  • (A) has score 2 * A, where A is a balanced parentheses string.
{% endblock question %} {% block solution_exp %}

Use open parentheses to determine the current score factor. Scores are added when a closed parentheses is reached only when the previous parentheses was open and the factor decreases with each closed parentheses.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int scoreOfParentheses(string S) { int score = 0; int levels = 0; bool goingIn = false; for (char c : S){ if (c == '('){ levels++; goingIn = true; } else { if (goingIn == true){ score += 1*pow(2,levels-1); goingIn = false; } levels--; } } return score; } }; {% endblock solution_code %}

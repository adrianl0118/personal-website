Given a non-empty, singly linked list with head node head, return a middle node of linked list.

If there are two middle nodes, return the second middle node.

{% endblock question %} {% block solution_exp %}

To find the midpoint of the list, traverse the entire list once to determine its length, then traverse again up to the midpoint and return.

{% endblock solution_exp %} {% block solution_code %} class Solution { public: ListNode* middleNode(ListNode* head) { int mid = getLen(head, 0)/2; while (mid > 0){ head = head->next; mid--; } return head; } int getLen(ListNode* head, int depth){ if (!head) return 0; if (head->next) return getLen(head->next, depth+1); else return depth+1; } }; {% endblock solution_code %}

Alex and Lee play a game with piles of stones. There are an even number of piles arranged in a row, and each pile has a positive integer number of stones piles[i].

The objective of the game is to end with the most stones. The total number of stones is odd, so there are no ties.

Alex and Lee take turns, with Alex starting first. Each turn, a player takes the entire pile of stones from either the beginning or the end of the row. This continues until there are no more piles left, at which point the person with the most stones wins.

Assuming Alex and Lee play optimally, return True if and only if Alex wins the game.

Memoization as described below.

{% endblock solution_exp %} {% block solution_code %} class Solution { private: vector> maxturnscore; //memoization tool, must have start and end info, the bounds of array public: bool stoneGame(vector& piles) { maxturnscore = vector>(piles.size()+1,vector(piles.size()+1,-1)); int total = accumulate(piles.begin(),piles.end(),0); int alexscore = maxscore(piles,0,piles.size()-1); if (alexscore > total-alexscore) return true; else return false; } //Calculate maximum number of stones that can be scored given any range of stone piles denoted by start, end int maxscore(vector& piles, int start, int end){ if (start >= end) return 0; if (maxturnscore[start][end] != -1) return maxturnscore[start][end]; //Calculate the current player's score by taking pile from this turn added to score from next turn for //both cases where current player takes front or back and where in the next turn the next player takes //front or back--take min of possible next turn scores of the current player because we assume the other player plays optimally. int frontscore = piles[start] + min(maxscore(piles,start+2,end), maxscore(piles,start+1,end-1)); int backscore = piles[end] + min(maxscore(piles,start, end-2),maxscore(piles,start+1,end-1)); return maxturnscore[start][end] = max(frontscore,backscore); } }; {% endblock solution_code %}

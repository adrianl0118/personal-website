Given the root node of a binary search tree, return the sum of values of all nodes with value between L and R (inclusive).

The binary search tree is guaranteed to have unique values.

{% endblock question %} {% block solution_exp %}

{% endblock solution_exp %} {% block solution_code %} class Solution { public: int rangeSumBST(TreeNode* root, int L, int R) { if (!root) return 0; int sum = rangeSumBST(root->left,L,R) + rangeSumBST(root->right,L,R); if (root->val >= L && root->val <= R) sum += root->val; return sum; } }; {% endblock solution_code %}

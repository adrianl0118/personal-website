A binary tree is univalued if every node in the tree has the same value.

Return true if and only if the given tree is univalued.

{% endblock question %} {% block solution_exp %}

Make code progress to increasingly deeper checks of the tree:

  1. Check NPE at the root node
  2. For each child node A) Check if the child node exists, B) Check whether its value is the same as the root's, C) Check if the branches constitute a unival tree
  3. If any of the checks in 1-2 returns false, cut off protocol immediately by returning false, without diving deeper into the tree needlessly
{% endblock solution_exp %} {% block solution_code %} class Solution { public: bool isUnivalTree(TreeNode* root) { if (!root){ return true; } if (root->left){ if (root->val != root->left->val || !isUnivalTree(root->left)){ return false; } } if (root->right){ if (root->val != root->right->val || !isUnivalTree(root->right)){ return false; } } return true; } }; {% endblock solution_code %}

Some of the things I've been working on--if there is a hyperlink embedded in the heading, it will take you straight to the repository on Github.

invoice-editor - app that calculates totals from editable line item quantities and prices (web app - front end design)

A simple invoice editor made with ReactJS, Material UI and Redux, supported by Webpack and Babel. Redux reducers were tested using Jest. The app allows users to add line items and calculates the subtotal, tax and total from the quantities and prices of all items. The quantity and price properties of line items can be edited after they have been submitted. Try the app here


declutter - cloud-hosted service for putting up and picking up used household items (android app)

Android application that hosts free personal ads for donating furniture or other household items and also enables users to find ads for desired items by area. App written in Kotlin and data is stored in the cloud on Google's Firebase. Some information regarding the motivation behind designing this app can be found in the related blog post.


tictactoe - web hosted game utilizing dynamic memory and rendering components from React JS (web app)

A simple React web application for playing tic tac toe with another human. Dynamically stores game states to allow for undoing of one or multiple turns. Play the game here.


quantum-dice - random number generator that leverages the power of quantum computers (web app)

Web application that produces random numbers with optional Dungeons and Dragons style adjustments. Uses quantum computers that are available to the public from IBM's quantum computing division. The related blog post has some other background information, and the app can be tried directly here.


workout_dj - voice controller for youtube (desktop app)

A simple desktop tool for controlling Youtube using voice commands through a miscrophone. Automatically skips ads and autoplays songs. Written in Python and uses Google's Speech Recognition library. Check out the related blog post here.


asiaeast - trip itinerary planner (android development)

A mobile tool that groups location data by geographical proximity and presents it with Kotlin. Queries a real-time Firebase Firestore database of geopoint data and outputs results to Google Calendar.


personal website and blog (web development)

This website! Built in Flask using Python with Bootstrap as indicated on the homepage. Hit the heading link for more details.


structify - college semester scheduler (android development)

Mobile app in Java that generates study/work schedules for up to 8 simultaneous courses over a college semester using course syllabi information. Renders a detailed Google Calendar-style GUI and outputs to Google Calendar.


translation business website (web development)

A simple static website for a friend's Chinese/English translation business made with Flask framework for Python and a Bootstrap 4 front end.


fastrecipe - online recipe finder (android development)

A simple mobile app that queries web address data stored in a SQLite database based on user inputs through a dynamic UI. Code based in Java.
